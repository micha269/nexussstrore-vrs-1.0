# Nexus Store vrs-1.0 🛒

Proyecto académico de desarrollo de un sistema de inventario de bodega con un chat de IA integrado.

## 🚀 Tecnologías utilizadas
* **Backend:** Laravel (PHP)
* **Frontend:** React (JavaScript)
* **Base de Datos:** MySQL (o la que estés usando)
* **Integraciones:** API de Chat con IA

## 🛠️ Instalación local

### Backend
1. Entra a la carpeta `backend/nexusstore-api`.
2. Ejecuta `composer install`.
3. Configura tu archivo `.env`.
4. Ejecuta `php artisan migrate`.

### Frontend
1. Entra a la carpeta `frontend/nexusstore-client`.
2. Ejecuta `npm install`.
3. Inicia el proyecto con `npm run dev`.

## 📄 Descripción
Este sistema permite gestionar productos, stock y ventas, ofreciendo una experiencia mejorada mediante un asistente virtual inteligente.
